üìã Troubleshooting Guide

If images are not loading:

  1. Check file names: Make sure files exist in the images folder with exact names
  2. Case sensitivity: GitHub Pages is case-sensitive, use lowercase filenames
  3. File format: Ensure images are in web-compatible formats (PNG, JPG, SVG)
  4. Path format: Use relative paths without ./ prefix for GitHub Pages
  5. Cache: Hard refresh browser (Ctrl+F5) to clear cache

GitHub Pages Specific Tips:

  • Allow 5-10 minutes for changes to propagate
  • Check the Actions tab for build errors
  • Verify files are in the correct repository structure
  • Test locally before deploying
